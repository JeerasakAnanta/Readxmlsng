```
https://github.com/JeerasakAnanta/Readxmlsng.git

```
## create virtual environment 
create 
```
python3 -m  venv env
``` 
activate
```                                        
source env_fastapi/bin/activate 

```

```
pip install -r requirements.txt  
pip install "fastapi[standard]"
```

```
fastapi run  api.py --reload
```
